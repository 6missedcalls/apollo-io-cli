### Get a List of Fields

Source: https://docs.apollo.io/reference/get-a-list-of-fields

Retrieves information about all fields present in your Apollo account. This endpoint mandates the use of a master API key for successful execution.

```APIDOC
## GET /websites/apollo_io/fields

### Description
Use the Get a List of Fields endpoint to retrieve information about all of the fields that exist in your Apollo account. This endpoint requires a master API key. If you attempt to call the endpoint without a master key, you will receive a `403` response. Refer to <a href="https://docs.apollo.io/docs/create-api-key" target="_blank">Create API Keys</a> to learn how to create a master API key.

### Method
GET

### Endpoint
/websites/apollo_io/fields

### Parameters
#### Query Parameters
- **api_key** (string) - Required - Your master API key.

### Request Example
```json
{
  "api_key": "YOUR_MASTER_API_KEY"
}
```

### Response
#### Success Response (200)
- **fields** (array) - A list of field objects, each containing details about a specific field.
  - **id** (string) - The unique identifier for the field.
  - **name** (string) - The name of the field.
  - **type** (string) - The data type of the field.

#### Response Example
```json
{
  "fields": [
    {
      "id": "field_123",
      "name": "Industry",
      "type": "dropdown"
    },
    {
      "id": "field_456",
      "name": "Company Size",
      "type": "number"
    }
  ]
}
```

#### Error Response (403)
- **error** (string) - Indicates that a master API key is required.

#### Error Response Example
```json
{
  "error": "Master API key is required to access this endpoint."
}
```
